# Phase 1 Blog Post

## Topic ideas

- Using context
- Different ways of writing functions (which also relates to context)

```javascript
const titleCased = () => {
  return true
}
```

```javascript
function titleCased(){
  return true
}
```

```javascript
const titleCased = function(){
  return true
}
```

```javascript
(function() {
  return true;
})();
```
